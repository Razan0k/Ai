# Safe AI Use Simulation

موقع عربي باتجاه RTL يعرض مقترح محاكاة توعوية حول الاستخدام الآمن للذكاء الاصطناعي داخل بيئة عمل حكومية.

## المتطلبات

- Node.js 20 أو أحدث
- npm

## التشغيل المحلي

```bash
npm install
npm run dev
```

ثم افتح الرابط الذي يظهر في الطرفية.

## بناء نسخة الإنتاج

```bash
npm run build
npm run preview
```

## التقنيات

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- lucide-react
